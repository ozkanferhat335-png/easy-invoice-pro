## 📋 Özellikleri

- ✅ **E-Fatura Entegrasyonu** (UBL-TR 2.1)
- ✅ **KDV Bildirimi Otomasyonu** (GİB Excel)
- ✅ **Fatura Yönetimi** (Kes, İzle, Raporla)
- ✅ **Müşteri & Ürün Yönetimi**
- ✅ **Lisanslama Sistemi** (FREE/PRO/ENTERPRISE)
- ✅ **SQLite Şifreli Veritabanı**
- ✅ **PDF & Excel Raporları**
- ✅ **Türkçe Arayüz** (100%)
- ✅ **Barkod Okuyucu Desteği**
- ✅ **QR Kod Üretimi**

## 🎯 Hedef Kitle

- Esnaf, Serbest Çalışan
- Muhasebeci, KOBI
- E-Fatura Kullanan Şirketler

## 🔧 Teknik Özellikler

- **Platform:** Windows Forms (.NET Framework 4.7.2)
- **Dil:** C# 7.3
- **Veritabanı:** SQLite (Şifreli)
- **UI Framework:** Bunifu UI WinForms
- **Boyut:** < 10MB (Installer)
- **Lisans Modeli:** 30 Gün Demo + Premium Paketler

## 📦 Kurulum

Bilgisayarınızda aşağıdakilerin yüklü olduğundan emin olun:

- Windows 7 veya üstü
- .NET Framework 4.7.2
- Visual Studio 2019+ (Geliştirme için)

## 🚀 Hızlı Başlangıç

1. Repository'i klonlayın
2. `EasyInvoicePro.sln` dosyasını açın
3. NuGet paketlerini geri yükleyin
4. F5 ile çalıştırın
